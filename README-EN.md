### Description

**ExifTool GUI** is a powerful application for reading and removing metadata from over 150 file types. It supports EXIF, IPTC, XMP, GPS data, ICC profiles, ID3 tags, and numerous proprietary camera formats—making it an essential tool for photographers, videographers, archivists, and data security professionals.

ExifTool GUI is a graphical user interface (GUI) wrapper around the command-line utility **[ExifTool](https://exiftool.org/)** developed by Phil Harvey.

---

<details>
<summary><b>Program Features</b></summary>

- Read and remove metadata from images, videos, audio files, and documents  
- Support for 150+ file formats and multiple metadata standards, including EXIF, IPTC, XMP, GPS, JFIF, GeoTIFF, ICC Profile, Photoshop IRB, FlashPix, AFCP, ID3, and more  
- Batch processing of files  
- Renowned for its accuracy, stability, and ability to handle rare or non-standard formats  
- Drag & Drop support and Windows context menu integration  
- Light/dark theme support  
- Add both individual files and entire folders (recursively)  
- View metadata **before cleaning**, **after cleaning**, and **what was removed**  
- Built-in lightweight metadata editor (for R/W-supported formats)

</details>

<details>
<summary><b>Version 1.1</b></summary>

---

#### 1. "Quick Clean" and "Clear Completed" Buttons

![Quick Clean](https://i.ibb.co/gbpJXBBz/2026-01-19-172449-compressed.jpg)

- **"Quick Clean"** removes metadata without prior scanning. Previously, users had to **Scan → Clean**. With Quick Clean, the original metadata count is not shown.  
- **"Clear Completed"** removes only successfully cleaned files from the list.

---

#### 2. New Columns: "Size", "Extension", "What Was Removed?"

![New Columns](https://i.ibb.co/4R0srN4T/2026-01-19-173044-compressed.jpg)

The **"What Was Removed?"** column shows exactly which metadata fields were deleted.

---

#### 3. Progress Bar & File Statistics

![Progress](https://i.ibb.co/SD2sg3Jp/2026-01-19-1713423-compressed.jpg)

A progress bar now appears during scanning and cleaning. The interface also displays:
- Total number of added files  
- Combined file size

---

#### 4. Customizable Column Visibility

![Column Settings](https://i.ibb.co/ZqBtLWd/2026-01-19-174412-compressed.jpg)

Users can hide or show any column for a cleaner view.

---

#### 5. Built-in Metadata Editor

![Editor](https://i.ibb.co/QFXnzwYN/2026-01-19-175147-compressed.jpg)

A lightweight editor allows editing metadata in formats that support **read/write (R/W)** operations via ExifTool.

---

#### 6. Reduced Application Size

Distribution size reduced by **35%** — from **60 MB → 39 MB**.
